## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/vanshika3016/AutoVerse.git
```

Navigate into the project

```bash
cd AutoVerse
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
